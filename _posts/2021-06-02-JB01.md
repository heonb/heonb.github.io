# 주피터 노트북 사용법
* shift+enter 키 누르면 셀 실행되고 커서가 다음셀로 이동함
* etner키 누르면 편집상태로 다시 돌아옮
* esc키 누르고 
    * a 누르면 위에 셀 추가
    * b 누르면 아래에 셀 추가
    * dd 누르면 셀 삭제
    * m 누르면 문서셀로 변경
    * y 누르면 코드셀로 변경
    * h 누르면 다양한 단축키 확인 가능함
* 되돌리고 싶을 경우, Edit-> undo delete cells 클릭하면 됨
